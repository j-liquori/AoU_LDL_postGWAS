# AoU_LDL_postGWAS
Post-GWAS analysis scripts for "Sex differences in LDL-C genetic architecture and statin efficacy in All of Us." Includes annotation, sex-stratified effect size comparisons, and visualization of GWAS summary statistics generated using REGENIE v2.2.4 from the All of Us Research Program (v7).

## Data Availability
The GWAS summary statistics used in this analysis are available on Zenodo:  
[[DOI](https://doi.org/10.5281/zenodo.19333987)](https://doi.org/10.5281/zenodo.19333987)

This repo is currently a work in progress.
